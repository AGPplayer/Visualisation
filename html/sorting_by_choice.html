Sorting by choice

At each step, the algorithm searches for the smallest element in the not-yet-sorted part of the array. The found minimum is reversed with the first element of this part. Thus, the array is gradually filled with the correct values from left to right.

  • Difficulty:

    Always, even if the array is already almost sorted.

  • Plus:

    Performs a minimum number of permutations, which is useful if writing to memory is "expensive".

  • When to use it:

    Only for educational purposes or for very small arrays where speed is not critical.
