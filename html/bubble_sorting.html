Сортировка пузырьком

Алгоритм проходит по массиву, сравнивая соседние элементы. Если текущий элемент больше следующего, они меняются местами. В результате каждого прохода самый большой элемент "всплывает" в конец списка, подобно пузырьку воздуха в воде.

  • Сложность:

    В любом случае квадратичная. Поэтому он медленно работает большими наборами данных.

  • Плюсы:

    Простота реализации и минимальные затраты памяти.

  • Когда использовать:

    Только для образовательных целей или для очень маленьких массивов, где скорость не имеет решающего значения.
