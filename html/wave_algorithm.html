Wave Algorithm

Wave Algorithm (also known as Lee algorithm) is a grid-based method for finding the shortest path between two points. The process begins at the starting cell. The algorithm then "spreads" like a wave to all adjacent neighbors. This expansion continues layer by layer until the target cell is reached or no more moves are possible. Finally, the shortest path is reconstructed by retracing the steps from the target back to the start in descending order.

  • Difficulty:

    The time and space complexity are both O(N × M), where N and M are the dimensions of the grid.

  • Plus:

    It is simple to implement and always guarantees finding the shortest path if one exists.

  • When to use it:

    Use it for pathfinding in mazes.
